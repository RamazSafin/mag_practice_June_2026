# Задание 2.2 — Раздаточные материалы

## Методика и технологии дистанционного обучения для корпоративного обучения

---

### Модели дистанционного обучения в корпоративной среде

| Модель | Описание | Преимущества | Ограничения |
|---|---|---|---|
| Самостоятельное изучение (on-demand) | Курс в LMS, самостоятельное прохождение, автоматический контроль | Гибкость, массовость, экономия времени преподавателя | Низкая вовлеченность, формальное прохождение |
| Смешанное обучение (blended) | Комбинация электронных модулей и очных/синхронных занятий | Сочетание самостоятельности и взаимодействия | Сложность организации расписания |
| Вебинарное обучение | Регулярные онлайн-занятия с элементами интерактива | Вовлеченность, обратная связь | Зависимость от технической инфраструктуры |
| Наставническое обучение | Электронный курс + сопровождение наставника/тьютора | Индивидуализация, практическая поддержка | Ресурсоемкость |

---

### Структура электронного модуля в LMS

1. **Цель и задачи модуля** — что должен знать и уметь обучающийся после прохождения.
2. **Видеолекция** (10–15 минут) — изложение основного содержания.
3. **Презентация** — визуальное сопровождение видеолекции для самостоятельного изучения.
4. **Текстовая инструкция** — нормативный документ, алгоритм, регламент.
5. **Контрольные вопросы** (10–15 вопросов) — проверка усвоения теоретического материала.
6. **Практическое задание** — кейс, сценарий, задача, связанная с профессиональной деятельностью.
7. **Условие завершения** — просмотр всех материалов + тест не менее 80%.

---

### Формы контроля в электронной среде

| Форма | Описание | Пример |
|---|---|---|
| Тест с автоматической проверкой | Вопросы с выбором ответа, соответствием, множественным выбором | Тест по охране труда, 15 вопросов |
| Практическое задание с загрузкой файла | Кейс или задача, решение загружается в LMS | Разработка плана эвакуации |
| Интерактивное задание (H5P) | Встроенный в страницу интерактивный элемент | Видеолекция с контрольными вопросами |
| Проектная работа | Разработка учебного материала, инструкции, презентации | Создание инструкции по СИЗ |
| Вебинар с обратной связью | Онлайн-занятие с вопросами, чатом, опросами | Консультация по итогам модуля |

### Цифровые инструменты для организации дистанционного обучения

| Инструмент | Назначение | Пример использования |
|---|---|---|
| Moodle LMS | Платформа электронного обучения, размещение курсов, тестирование, учет | Основная среда корпоративного курса |
| Nextcloud | Облачное хранение материалов | Хранение инструкций, регламентов, видеоматериалов |
| Jitsi Meet | Видеоконференции, вебинары | Онлайн-консультации, демонстрация экрана |
| Telegram (или Mattermost) | Оперативная коммуникация | Уведомления, вопросы, каналы по темам |
| H5P | Создание интерактивных элементов | Видеолекции с встроеенными вопросами |
| Mentimeter | Интерактивные опросы | Сбор обратной связи во время вебинара |

### Задание для групповой работы

**Разработайте фрагмент электронного модуля для корпоративного курса.**

Для вашей темы определите:

1. **Цель модуля** — формулируйте через результат (умение/знание).
2. **Перечень учебных материалов** — что будет изучать обучающийся.
3. **Практическое задание** — описание ситуации и критерии выполнения.
4. **Контрольный тест** — не менее 5 вопросов с вариантами ответа.
5. **Критерий успешного завершения** — минимальный балл, обязательные элементы.

**Темы для групп:**
— Группа 1: «Основы охраны труда» (строительная компания).
— Группа 2: «Информационная безопасность» (IT-компания).
— Группа 3: «Клиентский сервис» (торговая сеть).
— Группа 4: «Работа с документами» (государственное учреждение).

**Форма представления:** таблица, заполненная по пунктам 1–5.

**Время на выполнение:** 20 минут.
